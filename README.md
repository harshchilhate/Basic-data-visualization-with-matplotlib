## Generating Data - Basic data visualization with matplotlib


This section focuses on learning the fundamentals of data visualization using Matplotlib.

### What I implemented
- Line plots and scatter plots
- Explicit data mapping (x vs y)
- Plot styling using built-in styles
- Axis labels, titles, and tick customization
- Custom colors and colormaps
- Programmatic saving of plots

### Key learning
I learned how raw data is transformed into meaningful visual representations and why correct data mapping and readability are essential for accurate interpretation.
